## Навыки и инструменты

- **python**
- **pandas**
- **numpy**
- **matplotlib**
- **scipy.stats**

## Общий вывод

Был проведён анализ данных, проверены некоторые гипотезы и сделаны выводу по приносимой пользователям с подпиской прибыли.
